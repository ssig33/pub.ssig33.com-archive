最近の macOS (Monterey/Ventura とか)で入力ソースを AquaSKK だけにする方法

  1. AquaSKK を入れる
  2. 標準日本語入力の英字のチェックを入れる
  3. すると「ABC」が消せるようになるので消す
  4. 標準日本語入力も消せるので消す

こんなふうになる。この時「AquaSKK 統合」だけを残すこともできるんだが、それをやると React で書かれた Web アプリなどで入力が不安定になる(モードの切り替えが適切に保存されずすぐに直接入力になってしまう)ので、上にはったスクショのように設定するのがよい。
