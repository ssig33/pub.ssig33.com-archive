Slackでhideした人を完全消滅させる方法

Slackを開発者モードで起動した上で、コンソール開いて、以下を実行するようにすると、hideした人の発言が完全に見えなくなって最強便利。


setInterval(() => {
  Array.from(document.querySelectorAll(".c-message_kit__gutter"))
    .filter(e => e.textContent.match(/This message is hidden/))
    .forEach(e => e.remove());
}, 500);

会話の流れが追いづらくなるなどの問題はあるけどまあそのあたりはやっていきましょう。
